# Backend

## Installation

```bash
npm install
```

## Development

```bash
# Start in development mode
npm run start:dev

# Start in debug mode
npm run start:debug
```

## Production

```bash
# Build
npm run build

# Start in production mode
npm run start:prod
```

## Testing

```bash
# Unit tests
npm run test

# E2E tests
npm run test:e2e

# Test coverage
npm run test:cov
```

## Code Quality

```bash
# Lint
npm run lint

# Format code
npm run format
```
